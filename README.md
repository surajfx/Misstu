# Duটি — তোমাদের দুজনের প্রাইভেট চ্যাট

## যা যা আছে
- শুধু দুজনের জন্য প্রাইভেট রিয়েল-টাইম চ্যাট (Firebase Realtime Database)
- টেক্সট মেসেজ এডিট ও ডিলিট করা যাবে ("এডিট করা" ট্যাগ দেখাবে)
- ছবি ও ভিডিও শেয়ার (Cloudinary দিয়ে আপলোড — তোমার surajfx.in তে যেটা আগে থেকেই ব্যবহার করছো)
- অনলাইন/অফলাইন স্ট্যাটাস, "শেষ কখন দেখা হয়েছে", টাইপিং ইন্ডিকেটর
- সুন্দর অ্যানিমেটেড রাতের আকাশের ব্যাকগ্রাউন্ড
- পিন কোড দিয়ে লক — শুধু তোমরা দুজনই ঢুকতে পারবে

## সেটআপ (৫ ধাপ)

### ১. Firebase প্রজেক্ট বানাও
- console.firebase.google.com এ যাও, নতুন প্রজেক্ট বানাও (অথবা তোমার surajfx2 প্রজেক্টও ব্যবহার করতে পারো)
- বাঁদিকের মেনু থেকে **Realtime Database** চালু করো (Firestore নয়, Realtime Database)
- Rules ট্যাবে গিয়ে এটা বসাও (শুধু লগইন করা দুজনের জন্য — সহজ রাখতে):
```json
{
  "rules": {
    "rooms": {
      "$room": {
        ".read": true,
        ".write": true
      }
    }
  }
}
```
  *(এটা বেসিক রুলস, পাবলিক ইন্টারনেটে যে কেউ URL জানলে ডেটা দেখতে পারবে। চাইলে পরে Firebase Auth যোগ করে আরও সিকিউর করা যাবে — বলো আমি করে দেব।)*
- Project settings -> General -> "Add app" -> Web app। যে config অবজেক্ট পাবে সেটা `config.js` ফাইলে `firebaseConfig` এর জায়গায় বসাও।

### ২. Cloudinary সেটআপ (ছবি/ভিডিওর জন্য)
- cloudinary.com এ তোমার existing অ্যাকাউন্টে ঢোকো (surajfx.in তে যেটা ব্যবহার করেছো)
- Settings -> Upload -> "Add upload preset" -> Signing Mode: **Unsigned** করে সেভ করো
- Cloud name ও preset name `config.js` তে বসাও

### ৩. দুজনের নাম ও পিন সেট করো
`config.js` ফাইলে:
```js
const CHAT_USERS = {
  a: { name: "তোমার নাম", avatar: "" },
  b: { name: "ওর নাম", avatar: "" }
};
const CHAT_PIN = "তোমাদের পছন্দের পিন";
```

### ৪. ডিপ্লয় করো
- এই ফোল্ডারটা GitHub Pages, Netlify, বা তোমার নিজের হোস্টিং (surajfx.in এর সাবডোমেইন, যেমন duti.surajfx.in) এ আপলোড করো
- সব ফাইল (index.html, style.css, app.js, config.js) একই ফোল্ডারে রাখতে হবে

### ৫. টেস্ট করো
- দুজনে আলাদা ফোন থেকে সাইটে ঢোকো, নিজের নাম বেছে পিন দিয়ে ঢোকো
- একজন মেসেজ পাঠালে অন্যজনের ফোনে সাথে সাথে দেখাবে

## ভিডিওর সাইজ নিয়ে
Cloudinary ফ্রি প্ল্যানে ভিডিও আপলোডের একটা লিমিট থাকে (সাধারণত ১০০MB পর্যন্ত)। বড় ভিডিওর জন্য একটু সময় লাগতে পারে আপলোড হতে — অ্যাপে প্রোগ্রেস বার দেখাবে।

## পরে যা যোগ করা যাবে
- ভয়েস মেসেজ
- মেসেজ রিয়্যাকশন (❤️, 😂 ইত্যাদি)
- Firebase Auth দিয়ে আরও শক্ত সিকিউরিটি
- পুশ নোটিফিকেশন (PWA)

কিছু ঠিকঠাক না চললে বা নতুন ফিচার লাগলে জানিও।
